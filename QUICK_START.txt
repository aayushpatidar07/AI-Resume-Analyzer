# QUICK START GUIDE

## üöÄ Get Started in 5 Minutes

### Prerequisites
- Python 3.8+ installed
- pip (comes with Python)

### Step 1: Open Command Prompt/PowerShell
Navigate to the project folder:
```powershell
cd "C:\Users\ASUS\Documents\AI Resume Analyzer & Job Matcher"
```

### Step 2: Create Virtual Environment
```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
```

**If you get an error about execution policy:**
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

Then run the activate command again.

### Step 3: Install Dependencies
```powershell
pip install -r requirements.txt
```

Wait for all packages to install (this may take 1-2 minutes).

### Step 4: Run the Application
```powershell
python app.py
```

You should see:
```
============================================================
AI Resume Analyzer & Job Matcher
============================================================
Starting Flask development server...
Navigate to: http://127.0.0.1:5000
============================================================
```

### Step 5: Open in Browser
- Click on: http://127.0.0.1:5000
- Or copy-paste the URL into your browser

### Step 6: Test the App
1. **Download a sample resume** (use any PDF)
2. **Click "Load sample job description"**
3. **Select your resume PDF**
4. **Click "Analyze Resume"**
5. **View your results!**

## üõë If Something Goes Wrong

### Error: "Module not found"
```powershell
pip install -r requirements.txt --force-reinstall
```

### Error: "Port 5000 already in use"
Edit `app.py`, find this line:
```python
app.run(debug=True, host='127.0.0.1', port=5000)
```
Change `5000` to `5001` (or any available port)

### Error: "PDF cannot be read"
- Ensure PDF is not corrupted
- Try a different PDF file
- Check file size (should be < 16MB)

### App not responding
- Check the terminal for error messages
- Close the app (Ctrl+C)
- Run it again with: `python app.py`

## üí° Tips & Tricks
